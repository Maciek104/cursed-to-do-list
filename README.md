# Cursed To-Do List
Aplikacja do zarządzania zadaniami z nawiedzonym, mrocznym klimatem. 

## Funkcjonalności
- ✅ Dodawanie i usuwanie zadań
- 📅 Przypisywanie dat wykonania
- 📝 Dodawanie opisów do zadań
- 🔍 Wyszukiwanie zadań
- 📊 Sortowanie po dacie i tytule
- ✏️ Edytowanie zadań klikając na nie
- ☑️ Oznaczanie zadań jako ukończone

## Technologia
HTML, CSS, JavaScript
